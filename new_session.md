# new session.

When i start up a new session.  Its nice to be sure that everything is working before i start. So I use this prompt.

```
1. Please load prevous session information. include your memory, workflow.md and handoff.md. So that understand where we were in our
   last session.  Do not start coding just verfy that you are up to speed.
2. Please examine what MCP servers you have enabled, ensure that the connection is working to all of them. If there is an issue stop and tell
   me what that issue is so that i may help.
```
